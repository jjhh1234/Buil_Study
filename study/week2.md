## 1. 객체란 무엇인가
+ 클래스 : 붕어빵 틀, 설계도
+ 객체 : 만들어진 붕어빵들 (슈크림 붕어빵, 팥 붕어빵, 피자 붕어빵)
+ 인스턴스 : 클래스로 부터 만들어진 객체
+ 클래스 선언 방식
```Car car = new Car();```

## 2. 필드
+ 객체 고유의 데이터를 저장하는 공간
+ 선언형태는 변수와 비슷하지만 변수는 아님

```타입 필드 = 초기값;```

```String str = "hello";```

## 3. 생성자
+ 객체 생성시 초기호 담당
+ 클래스 이름으로 되어있고 리턴 타입 없음

## 4. 메소드
+ 객체의 동작을 설정
+ 메소드 안에서 선언된 변수는 메소드 안에서만 존재 - 실행이 끝나면 사라짐
+ 접근제한자, 리턴타입, 메소드 이름으로 이루어짐
